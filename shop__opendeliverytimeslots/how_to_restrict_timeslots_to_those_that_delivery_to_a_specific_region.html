Shop / OpenDeliveryTimeSlots API

How to restrict timeslots to those that delivery to a specific region

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/open_delivery_time_slots

Orders must be placed in a timeslot. Each timeslot has a time window in which it will accept orders and a window in which the order will be delivered or available for collection. Timeslots will only be included in the list during the period they will accept orders.

Parameters

Name Description
region_id Restrict the open delivery time slots those delivering to this region
