Shop / Facet Groups API

Retrieving facets without restricting will return an error

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:id/facet_groups

You must restrict which facets are returned by supplying the requested types with the only_facets[] query parameter. The available facets types are: special, department, tag, brand_name, place_of_origin, place_of_origin_region, and price_range. Some facets require features to be enabled per shop or other configuration. If a facet is not enabled for a shop, the facet will be not be returned. If you are manually creating facets to search e.g. creating price ranges. You do not need to search for that facet type.

Parameters

Name Description
department_id Restricts search to the given department and its sub-departments.
keywords Keywords matched against line name, description, and other fields
facets One or more facet query-values obtains from the Facet Groups end-point to refine a search.
specials Restricts search to lines on special
only_facets only facets
