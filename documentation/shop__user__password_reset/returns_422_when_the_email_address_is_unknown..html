Shop / User / Password Reset API

Returns 422 when the email address is unknown.

POST https://api.myfoodlink.com/smart_shopping/v2/user_identity/password_resets

Parameters

Name Description
email_address Email address of user requiring a password reset
reset_url_template Template for the reset URL to direct the customer to in their e-mail. Must be a valid https URL and have {{token}} in it.
