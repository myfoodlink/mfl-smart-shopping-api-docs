Shop / OpenDeliveryTimeSlots API

Discover delivery timeslots for this shop that will accept orders now when the deliveries are closed

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/open_delivery_time_slots

All deliveries for this day have been closed so no timeslots are returned

Name Description
region_id Restrict the open delivery time slots those delivering to this region
