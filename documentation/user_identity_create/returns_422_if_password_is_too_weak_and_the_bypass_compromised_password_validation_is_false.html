User identity Create API

Returns 422 if password is too weak and the bypass_compromised_password_validation is false

POST https://api.myfoodlink.com/smart_shopping/v2/user_identity

Parameters

Name Description
user_identity[first_name] First Name
user_identity[last_name] Last Name
user_identity[email_address] Email Address
user_identity[password] Password
user_identity[bypass_compromised_password_validation] bypass_compromised_password_validation
user_identity[birth_date] Birth Date
user_identity[phone_number] Phone Number
user_identity[billing_address][lines] The lines of the billing address, typically the street address and company information, eg [\"Myfoodlink\", \"1/183 Maquarie St\"]
user_identity[billing_address][locality] The locality, town or suburb, of the billing address, eg Lenah Valley.
user_identity[billing_address][postal_code] The postal code of the billing address, eg 7008.
user_identity[billing_address][administrative_area] The administrative area of the billing address, eg TAS.
user_identity[billing_address][country_code] The iso country code of the billing address, eg AU.
