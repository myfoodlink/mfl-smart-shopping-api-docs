Shop / Orders / Lines API

Update basket contents

POST https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/orders/:order_id/lines

Anonymous current user can create a basket. PUT operations update only the mentioned products in a basket, they don't remove unmentioned lines

Parameters

Name Description
order[order_lines] Products and quantities requested by customer. Each one is a hash with product_id and quantity. Use quantity 0 to remove lines.
