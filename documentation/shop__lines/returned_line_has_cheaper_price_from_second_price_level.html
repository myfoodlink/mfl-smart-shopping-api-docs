Shop / Lines API

Returned line has cheaper price from second price level

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:id/lines

Parameters

Name Description
department_id Restricts search to the given department and its sub-departments.
embedded_search_id Restricts search to a pre-canned search configured on the server, e.g. this is used for promoted lines at checkout. Additional restrictions can be applied on top.
deal_set Restricts the search to a deal set. Value to be used is the "query_value" attribute in a deal set.
facets One or more facet query-values obtained from the Facet Groups end-point to refine a search.
keywords Keywords matched against line name, description, and other fields
page 1-indexed page number of page to return
page_size Number of lines to include on each page
sort_option_id Sort option obtain from the Sort Options end-point.
specials Restricts search to lines on special
sell_price_cents[minimum] Restricts search to lines costing at least this much.
sell_price_cents[maximum] Restricts search to lines costing at most this much.
product_ids Product Id is shared across all stores.
product_codes Search by GTIN or instore PLU.
