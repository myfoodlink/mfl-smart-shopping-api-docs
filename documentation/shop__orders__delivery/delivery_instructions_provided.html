Shop / Orders / Delivery API

Delivery instructions provided

PATCH https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/orders/current/delivery

Parameters

Name Description
order[time_slot_id] The id for the chosen timeslot
order[instructions] Delivery instructions for the order. e.g. "code for the gate is 1234". Can only be used for deliveries and not pickup
order[delivery_address][unit] Unit information for the address, e.g. 'Flat 1', or 'Level 3'
order[delivery_address][lines] The lines of the delivery address, typically the street address and company information, eg ["Myfoodlink", "1/183 Maquarie St"]
order[delivery_address][locality] The locality, town or suburb, of the delivery address, eg Lenah Valley.
order[delivery_address][postal_code] The postal code of the delivery address, eg 7008.
order[delivery_address][administrative_area] The administrative area of the delivery address, eg TAS.
order[delivery_address][country_code] The iso country code of the delivery address, eg AU.
