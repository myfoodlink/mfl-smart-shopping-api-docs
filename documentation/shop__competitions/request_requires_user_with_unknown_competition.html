Shop / Competitions API

Request requires user with unknown competition

POST https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/competitions/:competition_id/competition_entries

This controller previously always required a signed in session. It will still return a permission error when no session supplied for an unknown competition or a competition that does not allow retail entry

Parameters

Name Description
order_id order
options This is an array of all the keys for options the customer has accepted. This includes all the mandatory options such as terms and conditions and also optional options such as mail list signup. This should only include accepted keys
first_name first name
last_name last name
phone_number phone number
post_code post code
receipt_photo receipt photo
email email
birth_date birth date
