Shops API

Searching for a unknown locality and postcode will return no shops

GET https://api.myfoodlink.com/smart_shopping/v2/shops

Parameters

Name Description
store_number Only return stores with this store number. Store number is normally unique, so this will generally only return a single store.
locality Restrict to this locality. Both locality and postcode must be sent. Otherwise this is ignored.
postcode Restrict to this postcode. Both postcode and locality must be sent. Otherwise this is ignored.
fulfilment_type Search for pickup or delivery. This must be sent if locality and postcode are sent. It can have the values "pickup" or "delivery"
shop_role Search for shop role. It can have the value "gift_cards"
