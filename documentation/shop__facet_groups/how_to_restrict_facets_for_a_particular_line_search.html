Shop / Facet Groups API

How to restrict facets for a particular line search

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:id/facet_groups

Retrieve the facets that are valid for a particular line search by supplying the same parameters as the search.

Parameters

Name Description
department_id Restricts search to the given department and its sub-departments.
keywords Keywords matched against line name, description, and other fields
facets One or more facet query-values obtains from the Facet Groups end-point to refine a search.
specials Restricts search to lines on special
only_facets only facets
