Shop / User / Payment Services API

How to retrieve the payment service available to a user.

GET https://api.myfoodlink.com/smart_shopping/v2/shops/:shop_id/user/payment_services

In this scenario there are two payment methods configured for the store. A paystream one that is forbidden to be used with V2 and a Till one that is forbidden to be used with V1. The results will only contain the Till service
