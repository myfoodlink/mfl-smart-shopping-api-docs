User Identity / Email Check API

Checks the system to see if an email address should sign in or sign up

When email is matched to existing user identity (case insensitive).

GET https://api.myfoodlink.com/smart_shopping/v2/user_identity/email_checks

Email matching is case insensitive - a user with email example@myfoodlink.test exists.

Parameters

Name Description
email_address Email Address
